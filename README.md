# Education Performance & Resource Allocation Dashboard #

Project Overview

This project is an interactive Microsoft Power BI dashboard developed to analyze student academic performance, attendance, resource utilization, and teacher workload within an educational institution.

The dashboard helps institutions:

Monitor student performance
Track attendance trends
Analyze resource usage
Evaluate teacher workload
Improve academic resource allocation

Features:
Student performance analysis by subject
Attendance tracking dashboard
Teacher workload monitoring
Resource utilization analytics
Interactive visual reports
Year and quarter-wise filtering
Detailed student-level data table

Dashboard Components:
KPI Metrics
Total Attendance
Total Workload Hours
Classes Assigned
Resource Usage Hours
Visualizations
Attendance Gauge Chart
Workload Gauge Chart
Subject-wise Attendance Bar Chart
Resource Usage Waterfall Chart
Teacher Workload Donut Chart
Subject Analytics Flow Visualization
Detailed Student Data Table

Technologies Used:
Microsoft Power BI
Microsoft Excel Dataset
DAX Measures
Data Modeling
Power Query
Dataset Information

The dataset contains:

Student Details
Subject Information
Attendance Records
Marks & Capacity
Teacher Assignments
Resource Usage Data
Workload Hours
Academic Year & Quarter

Objectives:
Improve educational performance monitoring
Optimize resource allocation
Support academic decision-making
Enhance teacher workload management
Identify low-performing areas

Key Insights:
Maths subject recorded the highest attendance.
Teacher workload distribution varies significantly.
Resource usage trends help identify utilization efficiency.
Student performance can be monitored effectively through centralized reporting.

Conclusion:
This dashboard provides a centralized and interactive solution for educational institutions to monitor academic performance, attendance, and operational efficiency using data-driven insights.


# HR Recruitment & Employee Performance Dashboard #

Project Overview

This project is an interactive Microsoft Power BI dashboard designed to analyze employee performance, recruitment costs, salary distribution, and hiring sources within an organization.

The dashboard enables HR teams to:

Track employee performance
Monitor recruitment expenses
Analyze salary distribution
Evaluate hiring channels
Improve workforce management

Features:
Employee performance tracking
Recruitment source analysis
Salary monitoring dashboard
Hiring cost visualization
Interactive employee analytics
Performance comparison reports
HR operational insights

Dashboard Components:
KPI Metrics
Total Salary
Total Recruitment Cost
Employee Performance Score
Hiring Source Contribution
Visualizations
Employee Performance Bar Chart
Salary Gauge Chart
Recruitment Cost Analysis
Hiring Source Breakdown
Employee Performance Donut Chart
Employee Summary Table

Technologies Used:
Microsoft Power BI
Microsoft Excel
DAX Functions
Power Query Editor
Data Visualization Techniques
Dataset Information

The dataset includes:

Employee Details
Salary Information
Recruitment Costs
Performance Scores
Job IDs
Hiring Sources
HR Metrics

Objectives:
Improve HR decision-making
Analyze employee performance trends
Monitor recruitment efficiency
Optimize hiring costs
Evaluate hiring channel effectiveness

Key Insights:
LinkedIn generated higher recruitment investments.
Employee performance scores vary across departments.
Salary distribution analysis helps HR budgeting.
Recruitment source comparison improves hiring strategies.

Future Enhancements:
Predictive employee attrition analysis
Real-time HR database integration
AI-powered hiring recommendations
Employee productivity forecasting
Advanced departmental analytics

Conclusion:
This HR dashboard delivers valuable insights into employee performance, salary distribution, and recruitment efficiency, helping organizations make informed HR and workforce management decisions.
